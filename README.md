# 🛡️ Vigia - Monitoramento & Logística

O **Vigia** é um aplicativo mobile desenvolvido em Flutter focado no monitoramento de rotas e gestão de logística em tempo real. O app oferece uma interface moderna com elementos neon, suporte total a modo escuro e rastreamento dinâmico de passageiros e veículos.

---

## 🚀 Funcionalidades

✨ **Dashboard Inteligente**
Visualização de estatísticas diárias (rotas, passageiros, atrasos e avaliações) com gráficos dinâmicos.

📍 **Monitoramento em Tempo Real**
Mapa interativo com acompanhamento de rotas via OSRM (Open Source Routing Machine).

🌙 **Interface Adaptativa**
Design responsivo para diferentes tamanhos de tela com suporte completo a **Dark Mode**.

🧭 **Navegação Fluida**
Animações suaves de rotas com rotação automática do ícone de navegação.

## 📦 Como rodar o projeto

### 🔧 Pré-requisitos

* Flutter SDK instalado
* Servidor OSRM rodando (local ou remoto)

### 📥 Clone o repositório

```bash
git clone https://github.com/seu-usuario/VigiaApp2.0.git
```

### 📂 Acesse a pasta

```bash
cd vigia
```

### 📦 Instale as dependências

```bash
flutter pub get
```

### 🎨 Gere os ícones do app

```bash
flutter pub run flutter_launcher_icons
```

### ▶️ Execute o projeto

```bash
flutter run
```
